# RS-014 Agent Rules Framework

## Goal
Define how project-specific AGENTS rules are authored and applied for this project and reusable in other projects.

## In Scope
- THIS-project rule profile expectations.
- OTHER-project transfer/process template expectations.
- Setup-time application behavior.

## Out of Scope
- Enforcing individual rule values globally.
- Runtime per-phase reapplication unless explicitly requested.

## Observable Outcomes
- Rule ownership and lifecycle are clear.
- Reuse process for external projects is understandable.
