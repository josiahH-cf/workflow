# RS-019 Cross-Review Suggestion Tone

## Goal
Define late-phase suggestion behavior so environment/window recommendations are minimal, optional, and non-disruptive.

## In Scope
- Suggestion frequency and tone guidance.
- Optionality expectations for new windows/terminals.

## Out of Scope
- Forcing environment isolation policies.
- Suppressing all late-phase suggestions.

## Observable Outcomes
- Late-phase prompts avoid noisy directional chatter.
- Useful suggestions remain available without pressure.

## Dependencies
- RS-012.
