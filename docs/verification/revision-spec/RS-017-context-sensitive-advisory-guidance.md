# RS-017 Context-Sensitive Advisory Guidance

## Goal
Define always-on advisory guidance behavior that adapts language based on branch/context signals while remaining non-blocking.

## In Scope
- Context signal detection goals.
- Advisory recommendation style expectations.
- Risk-aware prompts for blockers and routing suggestions.

## Out of Scope
- Mandatory routing enforcement.
- Exhaustive context classifier design.

## Observable Outcomes
- Guidance is more specific when contextual evidence exists.
- Suggestions remain optional and actionable.
