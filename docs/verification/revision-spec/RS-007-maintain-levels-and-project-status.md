# RS-007 Maintain Levels And Project Status

## Goal
Define project-wide maintenance-level semantics and status representation so maintenance is an ongoing state rather than terminal completion.

## In Scope
- Three maintenance levels.
- Project-wide status representation for maintenance state.
- Guidance for introducing new features while in maintenance mode.

## Out of Scope
- Implementing CI schedules.
- Forcing any single maintenance level by default.

## Observable Outcomes
- Maintain phase asks for level and records it clearly.
- Status language no longer implies maintenance is final completion.

## Dependencies
- None.
